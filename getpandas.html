Development Version¶

Stable windows binaries are built on a rotating basis every hour, as long as there have been code changes on github since the previous build. You can find the latest builds here.

Development is source controlled using git. Here’s a handy guide to getting started. If you’re interested in contributing to development, please visit our developer page.
